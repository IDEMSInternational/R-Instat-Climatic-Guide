19.1 Examining trends

Check on scale, daily, monthly, annual.

Consider adding broken stick – segmented regression model. With fixed or estimated break points. Package segmented may be used if estimated. Probably not needed if a break-point and want to try a fixed value.

See, for example https://stats.stackexchange.com/questions/382114/segmented-regression
